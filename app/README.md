# Distracted Driver Detection

## Prerequisites

* python 3.x
* PyQt5
* tensorflow
* numpy
* opencv-python
```sh
pip install pyqt5 tensorflow numpy opencv-python
```

## Usage

* Put images you want to test in `data/testImage`
* Run the program
```sh
python main.py
```
* Extracted images of video are in `data/testVideo/[video_name]`
